# Neural-Health-Predictor
IA aplicada à previsão de condições médicas leves com dados simulados
Stack: Python, Scikit-learn, Pandas, Matplotlib
Objetivo: Criar um modelo de machine learning que analisa dados vitais (pressão, glicemia, frequência cardíaca etc.) e prediz risco de uma condição simulada — por exemplo, “fadiga crônica” ou “hipertensão leve”
